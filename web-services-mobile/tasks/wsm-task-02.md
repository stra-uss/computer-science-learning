
#### Exercício 02

- __Descrição:__

O exercício consite em desenvolver uma aplicação front-end moderna e responsiva de um [CRUD] (# __(1) CRUD__) . Assim, o design deverá ser composto por uma abordagem limpa, ou seja, utilizando CSS com Flexbox/Grid. A aplicação, também, deverá, para ser responsiva, adaptar -se a celulares, tablets e desktops.
O escopo do exercício será o front-end, ou seja, a aplicação não precisará conter chamadas ao back-end. No entanto, deverão ser comentadas as respectivas chamadas.

A fim de simular um ambiente real de desenvolvimento de software moderno, a aplicação desenvolvida deverá ser hospedada na nuvem gratuita Vercel, utilizando o GitHub como repositório.


- __Passos necessários:__

 - 1. Criação de conta no GitHub [GitHub](https://github.com/)
  - (Tarefa realizada na Fase 1 da disciplina)

 - 2. Instalação da IDE VS Code
  - (Tarefa sugerida na Fase 1 da disciplina)

 - 3. Criação de uma conta na Vercel [Vercel Cloud](https://vercel.com/)

 - 4. Criação de um repositório no GitHub

 - 5. Associação do repositório criado (passo 4) à sua conta na Vercel

 - 6. Codificação (HTML, CSS e Javascript)

 - 7. Deploy na Vercel

 - 9. Execução da aplicação na internet


- #### __(1) CRUD__
  CRUD é um acrônimo para as quatro operações fundamentais que realizamos ao manipular dados (como em bancos de dados ou sistemas de software), sendo:
  - _Create_ (Cadastrar);
  - _Read_ (Ler);
  - _Update_ (Atualizar); e
  - _Delete_ (Deletar).
